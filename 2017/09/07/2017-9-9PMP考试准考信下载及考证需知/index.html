2017.9.9PMP考试准考信下载及考证需知

发表于 2017-09-07 | 更新于: 2018-10-18
字数统计: 1k | 阅读时长 ≈ 3

各位考生：

为保证参加9月9日项目管理考试的每位考生都能顺利进入考场参加考试，请完整阅读本通知内容。

一、关于准考信下载

为确保您顺利进入考场参加9月份考试，请及时登录http://exam.chinapmp.cn/ 个人系统下载并打印准考信，下载时间2017年9月6日-9月9日。如以上方式均无法查找准考信，请您及时联系我们：发送邮件chinapm@safea.gov.cn，或电话010-68413708、68940644或010-68948899转50424/50401

二、关于考场座位图和考场地址确认

请提前登录本网站个人系统查看您所在的考场安排信息。

北京地区考生除可在个人系统查看考场安排外，同时可点击“9月9日项目管理考试北京地区学员座位图”查看所在教室。

三、关于考场规定（含答题卡填写说明下载）

以下要求由PMI的监考机构普尔文公司（Prometric）提供，Prometric公司是美国教育考试服务院（ETS）的子公司，从事专业的考试开发、管理和执行，为保证考试公正，从不介入培训。Prometric公司将提供PMI认证考试的全程考试管理。

请务必遵守。

1**， 考试日期:** 2017年9月9日；PMP®, PBA和PgMP®考试时长: 4小时。ACP和CAPM®考试时长：3个小时。

报到时间: 上午8:00 (请考生准时到场, 不要迟到)

2**，证件要求：**

可接受的真实、有效身份证件和PMI准考信（两者缺一不可）。

可接受的真实、有效期内的身份证件类别：

l 身份证

l 护照

l 驾驶证

l 军官证

l 港澳台胞居留证或通行证

考生请注意身份证的有效期，到期的证件需要及时更换，过期证件不能参加考试。身份证更换和身份证丢失的考生需注意，公安局开具的身份证更换凭条不能作为身份证件参加考试。有效期内的临时身份证可以参加考试。

3**，允许带入考场的物品：**

有效身份证件，准考信，机械计时表

4**，禁止进入考场的物品：**

PMI明确要求不允许考生使用自带文具。自带的签字笔、铅笔，削笔刀等文具,眼镜盒，铅袋等均不可以带到座位上,因此考生在进入考场后将所以各人物品,自带文具统一存放在监考老师指定的物品存放处，监考机构将给每位考生提供自动铅笔，计算器和橡皮。电子表,iWatch和电子计时器也不允许携带到考场,考场提供钟表。考试中所有内容包括答题卡的所有填图均使用铅笔完成，考生在考试过程中如果需要更换铅芯或橡皮，请向监考老师举手示意。考试没有额外填图答题卡的时间,请考生将答案直接填图在答题卡上。

严禁在进入考场之后拿出或使用iWatch，手机、电子词典，录音笔,笔记，纸条及任何可扫描或拍照,录音录像的电子设备。如发现违规考生，将视为作弊停止其考试，并且记录在案。因此考生不要携带以上物品参加考试。并且请在进入考场之前关闭所有电子设备，并统一存放在监考老师指定的物品存放处。

5，答题卡填写说明和考场规则

见附件《考场规则和答题卡填写说明》，请仔细阅读。

对于考试规则和答题卡填写如有疑问请致电Prometric公司 010-82617799 。

附件：1.《考场规则和答题卡填写说明》-PMP®考试

2.《考场规则和答题卡填写说明》-ACP®考试
